Assembler:

assembler.l	: written in "flex"
lex.yy.c    	: C program generated from assembler.l


Compile lex.cc.yy (using gcc) to get 'a.out'.
./a.out takes in assembly code (format same as codes in folder "assembly_codes").
./a.out outputs in stdout the .coe file.
Error handling is done partially.
